# Space_Shooter

A small but structured **C++ arcade-style space shooter** built with **raylib** and **CMake**.
The project uses Raylib version 5.5, which is installed by its CMake config.

**Controls:** 
- W | ← : Move left
- D | → : Move right
- SPACE : Shoot

# Requirements:

- C++ compiler (Clang, GCC, or MSVC)
- CMake ≥ 3.15
- Git
- Raylib 5.5

# Clone the Project:

`git clone https://github.com/hirchmi19/Space_Shooter.git` <br>
`cd Space_Shooter`

# Build & Run:

`mkdir build`<br>
`cd build`<br>
`cmake -B build` <br>
`cmake --build build`

### Run the exe (example):

`./cmake-build-debug/Space_Shooter`
